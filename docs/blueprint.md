# **App Name**: EpochReads

## Core Features:

- Visitor Sign-in: Allow students, faculty, and employees to log their library visit by selecting a purpose ('Assignment', 'Use of Computer', 'Board games', 'Researching', 'Reading', 'Other') and entering their institutional email, with a confirmation message upon successful logging.
- Admin Authentication & Access Control: Provide a secure login for administrators using NEU institutional email to access administrative features and control user access.
- Real-time Visit Tracking & Dashboard: Display real-time and summary statistics for visitors (today, this week, this month), including graphical distributions (bar graphs) of visitors by college and visit purpose.
- Detailed Visitor Log & Search: Enable administrators to view, search, and filter detailed visitor logs by various criteria such as ID number, full name, email, role, program, reason, and visit time.
- User Account Management (Admin): Allow administrators to search, filter, sort, and manage user accounts (student, staff, faculty), with capabilities to block or delete users.
- Data Export Functionality: Provide an option for administrators to export filtered visitor log data in a common format for external reporting and record-keeping.
- AI Purpose Categorization: When a visitor selects 'Other' for their visit purpose and provides free-text, an AI tool will analyze the input to suggest or automatically assign a more specific category, aiding data standardization.

## Style Guidelines:

- Primary color: A vibrant orange, `#FF7D2D`, used for calls to action, interactive elements, and key highlights, providing an energetic and engaging feel.
- Background color: A deep blue-grey, `#233C4B`, serving as the primary dark backdrop, evoking professionalism, stability, and focus, suitable for an academic environment.
- Accent color: A warm yellow-orange, `#FAC846`, for secondary highlights, progress indicators, or complementary visual elements, adding clarity and approachability.
- Headline and body text font: 'arial' (sans-serif) for all text elements, providing a clear, legible, and widely accessible typography. Note: currently only Google Fonts are supported.
- Use official New Era University logo in specified locations, such as the welcome page and admin navigation sidebar.
- Admin dashboard layout features a persistent left-hand navigation sidebar with options like 'Dashboard', 'Visitor Log', and 'Account Management' alongside the NEU Library logo and user institutional email, promoting consistent navigation.
- A small green check pop-up will provide instant feedback upon successful visit logging for visitors.
